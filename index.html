Why being open source is important?

  • Open source software can be publicly audited by trusted security experts from all around the world.
  • It's much easier to put unnoticed backdoor in closed source software. Even if developers are trustable, they can be bullied to add the backdoor.
  • Someone targeted by the security hole can find what happened, since he/she have access to the sources.

Why being decentralized is important?

  • As seen in example of megaupload or silk road take down there is no place single in a world, where centralised server could live undisturbed.
  • It's much easier to block access to centralised server or take this server down.
  • Even if messages are encrypted the fact who talks to whom should be hidden as well. It's much easier to capture all metadata with centralised server.
